# ScrollToTop Component

## Descriere
Componentă utilitară (fără UI) care gestionează poziția de scroll la navigare.

## Funcționalități Cheie
*   **Resetare Automată**: Aduce pagina la început (0,0) la fiecare schimbare de rută.
*   **Integrare Router**: Ascultă hook-ul `useLocation` din `wouter`.
