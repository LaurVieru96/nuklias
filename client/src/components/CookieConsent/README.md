# CookieConsent Component

## Descriere
Sistem complet pentru gestionarea consimțământului cookie-urilor, conform normelor GDPR/ePrivacy.

## Funcționalități Cheie
*   **Banner Global**: Apare automat la baza ecranului pentru vizitatorii noi.
*   **Persistență**: Salvează preferințele în `localStorage`.
*   **Personalizare**: Modal dedicat pentru activarea/dezactivarea categoriilor de cookie-uri (Analitice, Marketing).
*   **Integrare GDPR**: Link direct către Politica de Confidențialitate.
