# ServiceCard Component

## Descriere
Componentă reutilizabilă pentru afișarea pachetelor de servicii (Pricing Cards).

## Funcționalități Cheie
*   **Tematică Dinamică**: Culori diferite în funcție de `tier` (Bronze - Sky Blue, Silver - Emerald Green, Gold - Purple).
*   **Dark Component**: Complet adaptat pentru Dark Mode.
*   **Animații**: Efecte de hover și intrare în pagină.
*   **Feature List**: Listă cu bife stilizate pentru fiecare beneficiu inclus.
