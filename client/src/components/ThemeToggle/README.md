# ThemeToggle Component

## Descriere
Buton pentru comutarea între Dark Mode și Light Mode.

## Funcționalități Cheie
*   **Iconițe Dinamice**: Soare pentru Light Mode, Lună pentru Dark Mode.
*   **Detectare Sistem**: Inițial preia preferința sistemului de operare.
*   **Persistență**: Ține minte alegerea utilizatorului.
