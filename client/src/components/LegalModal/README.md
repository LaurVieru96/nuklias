# LegalModal Component

## Descriere
Componentă de tip Dialog pentru afișarea textelor legale lungi (Termeni și Condiții, Politica de Confidențialitate).

## Funcționalități Cheie
*   **Scroll Area**: Zonă de text scrollabilă pentru conținut lung.
*   **Reutilizabil**: Primește titlul și conținutul ca props.
*   **Accesibil**: Bazat pe componenta `Dialog` din shadcn/ui.
