# Navbar Component

## Descriere
Bara de navigare principală a aplicației. Asigură accesul la paginile principale și se adaptează pentru dispozitive mobile.

## Funcționalități Cheie
*   **Logo Brand**: Link către Home.
*   **Navigare Desktop**: Link-uri sub formă de pastilă, cu evidențiere (Lime Green) pentru pagina activă.
*   **Navigare Mobil**: Meniu hamburger (drawer) animat cu `framer-motion`.
*   **Stare Activă**: Detectează automat ruta curentă și colorează butonul corespunzător.
