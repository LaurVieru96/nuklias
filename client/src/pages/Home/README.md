# Pagina Home (Acasă)

## Descriere
Pagina principală a aplicației, servind ca punct de intrare pentru vizitatori. Prezintă viziunea generală a agenției și serviciile oferite.

## Funcționalități Cheie
*   **Secțiune Hero**: Introducere captivantă cu titlu și Call-to-Action (CTA).
*   **Servicii**: Prezentarea pe scurt a pachetelor de servicii (Basic, Hosting, Growth) folosind carduri de prețuri.
*   **Beneficii**: De ce să alegi agenția noastră.
*   **Cookie Consent**: Integrare banner pentru consimțământ cookies (global).
