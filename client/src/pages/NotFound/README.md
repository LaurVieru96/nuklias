# Pagina Not Found (404)

## Descriere
Pagina afișată atunci când utilizatorul accesează o rută care nu există.

## Funcționalități Cheie
*   **Mesaj Clar**: "Pagina nu a fost găsită".
*   **Link Către Home**: Buton pentru a redirecționa utilizatorul înapoi la siguranță.
*   **Design Prietenos**: Evită frustrarea utilizatorului printr-un design plăcut.
