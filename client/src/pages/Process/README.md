# Pagina Process (Procesul Nostru)

## Descriere
Explică metodologia de lucru a agenției, pas cu pas, pentru a oferi transparență clienților.

## Funcționalități Cheie
*   **Timeline Vizual**: Prezentarea etapelor proiectului (ex: Onboarding, Strategy, Execution).
*   **Design Clean**: Focus pe conținut text lizibil și pași logici.
*   **Theme Aware**: Adaptat perfect pentru Dark Mode și Light Mode.
